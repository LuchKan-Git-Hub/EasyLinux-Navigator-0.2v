# EasyLinux Navigator | 0.2v

A beginner-friendly Linux terminal navigator and open-source Python project.

EasyLinux Navigator is designed to help newcomers explore and manage files and directories in Linux without memorizing commands. It allows you to:

- **View the current working directory**
- **List files and folders**
- **Create folders or text files**
- **Move between directories and save favorite paths for quick access**

But it’s not just a helper tool—it’s also an **open-source Python project**! The code is clean, modular, and easy to read, making it a great resource for anyone learning Python, file system operations, and basic scripting practices.

Whether you’re learning Linux or Python, EasyLinux Navigator gives you a hands-on way to experiment with real code while improving your terminal skills.
